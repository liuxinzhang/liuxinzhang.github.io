CSS总结

发表于 2020-02-01 更新于 2022-08-25 分类于 前端

总结：

  1. html引入css的三种方式：行内>内嵌>外联
  2. 元素的三种显示模式：行内、行内块、块级
  3. 常用的css属性：文本属性+背景属性+盒子属性+浮动属性+定位属性+装饰属性
  4. css三大特性：继承性，优先性，层叠性
  5. css选择器：基本选择器（标签、id、class、*），复合选择器（后代、子代、并集、交集、伪类）、结构伪类、伪元素选择器（了解）
  6. 盒子模型：组成（内容+内边距+边框+外边距），自动内减
  7. 浮动：浮动作用、浮动特点、浮动影响、清除浮动影响
  8. 定位：定位作用、定位方式、定位层级关系

共识：

  • 标签内容默认在盒子左上方，table td单元格默认垂直居中，水平左对齐
  • 盒子原点默认在左上方
  • 伪类的意思是假的类，或者说没有写出来的类，它描述了一些具有相同特征的元素，如具有某种状态的元素
  1. html引入css的三种方式
    1. 行内：标签属性位置编写style="属性:属性值;...";
    2. 内嵌：<style>css代码<style>，通常写在head标签里
    3. 外联：<link ref="stylesheet" href="css路径"></link>
  2. 元素显示模式
    1. 行内块元素（inline-block）：一行可以有多个；宽高由内容撑开；可设置宽高。img、input、button、textarea、select
    2. 行内元素（inline）：一行可以有多个；宽高由内容撑开；设置框高无效。a、span
    3. 块级元素（block）：独占一行；宽度默认同父元素，高度由内容撑开；可设置宽高。如div、p、li、h
    4. 元素显示模式转换：display:模式;
  3. css属性
    1. 字体和文本属性：
      1. color：字体颜色
      2. font-size：字体大小
      3. text-align：水平居中，可作用于行内块和行内标签。水平居中属性必须写在有宽的盒子才有效（没宽我怎么居中）。
      4. text-decoration：文本修饰。常用text-decoration:none;去除a标签默认下划线
      5. line-height：行高。文本默认都有行高（大概1.5倍）
        1. 取值方式：① 数字+px；② 倍数（当前标签font-size的倍数）
        2. 常用line-height:文本父元素高度;进行文本垂直居中
    2. 背景属性：background
      1. ～-color：背景色
      2. ～-image：背景图。url(‘图片路径’)，背景图属于装饰属性不回撑开盒子
      3. ～-repeat：平铺方式。默认会垂直平铺和水平平铺，常用属性值no-repeat。
      4. ～-position：位置。
    3. 盒子属性：
      1. border：边框。border: 1px solid red;
      2. padding：内边距。padding: 0 auto;版心居中
      3. margin：外边距。
      4. 边距快捷记忆：从上开始，顺时针赋值，无值看对面
    4. 浮动属性：
      1. float：浮动。float:left;
      2. overflow：清除浮动。overflow:hidden;
    5. 定位属性：
      1. position：取值：relative（相对）、absolute（绝对）、fixed（固定）
      2. 相对于参照物偏移：上下左右
    6. 装饰属性：
      1. 隐藏属性：
        1. visibility: hidden;不脱标占位
        2. display: none;脱标不占位（常用）
      2. overflow：溢出部分显示效果。取值：hidden-隐藏、visible-显示（默认）
      3. border-raidus：盒子圆角边框，正方形的话属性值写50%会变成圆
      4. vertical-align：垂直对齐方式，baseline-基线对齐（默认）、middle-中部对齐，top、bottom，常用于处理文字和行内块对齐、行内块和行内块对齐。浏览器遇到行内和行内块标签当做文字处理，默认文字是按基线对象
  4. css三大特性：最终原则：就近原则、最小范围原则
    1. 继承性
      1. 大部分文本属性都能被继承
      2. 自己有的属性用自己的，有些标签有缺省的默认样式，不会用继承（就近）
    2. 优先性
      1. 优先级公式（最小范围）：继承<通配符选择器<标签选择器<class选择器<id选择器<行内样式<!important
      2. 权重叠加计算公式：（行内样式的个数，id选择器的个数，类选择器的个数，标签选择器的个数），先判断选择器最终是否选择到了对应元素（没有就是继承，优先级最低），再根据公式计算
      3. !important无法提升继承属性的优先级，!important非继承时优先级最高
      4. 如果说都是继承，则最近父元素样式的优先级最高（就近）
    3. 层叠性
      1. 同一个标签的不同样式：叠加；同一标签优先级一样的相同样式：覆盖（后写的覆盖先写）
      2. 同一标签的相同样式生效规则：先按优先性，再按层叠性
  5. css选择器
    1. 基本选择器
      1. 标签（元素）选择器：标签 {}
      2. class选择器：.类名 {}
      3. id选择器：#id {}
      4. 通用选择器：* {}，可以理解成很多个元素选择器
    2. 复合选择器：一系列选择器组合使用，可以嵌套多层
      1. 后代选择器：选择器1 选择器2 {}
      2. 子代选择器：选择器1>选择器2 {}
      3. 并集选择器：选择器之间逗号隔开，。选择器1,选择器2 {}
      4. 交集选择器：同时满足多个选择器的标签（如div.box：既是div标签有属于box类），选择器之间紧挨着，有标签选择器时必须写在最前面（不然区分不开，因为紧紧的连在了一起）。选择器1选择器2 {}
      5. 伪类选择器：选择标签的某个状态。选择器:hover {}
    3. 结构伪类选择器：根据元素的结构关系选择元素，常用于选择父元素的子元素。图示看最下方图1
      1. 结构表达式：E:last-child，E:first-child，E:nth-child(n)，E:nth-last-child(n)
      2. n常见公式（n从0开始）：偶数2n，奇数2n+1，前5个-n+5，第五个开始n+5
  6. 盒子模型
    1. 盒子模型组成：内容（标签的宽高默认指的是内容的框高）、内边距（padding）、边框（border）、外边距（margin），图看最底下图2
    2. 自动内减：自动把内容是xy的盒子变成内容+内边距+边框是xy的盒子，设置box-sizing: border-box;属性即可启动自动内减
    3. 外边距塌陷现象（了解）
      1. 兄弟塌陷：垂直块级元素上下margin会塌陷，以较长的外边距为最终值；解决方案：只给一个元素设置margin
      2. 父子塌陷：互相嵌套的块级元素，子元素的margin-top会作用在父元素上；解决方案：① overflow: hidden ② 改为行内块
  7. 浮动
    1. 为什么使用浮动，使用行内块布局不行吗？
      1. 浏览器解析行内块或行内标签的时候，如果标签换行书写会产生一个空格的距离
    2. 浮动作用：垂直布局的块级元素 转为 水平布局，且不留空隙
    3. 浮动特点：
      1. 浮动后会脱标，不占用标准流
      2. 浮动后元素显示模式变为行内块
      3. 浮动后text-align: center;，margin: 0 auto;自动居中失效
    4. 浮动影响：父子级标签，子级浮动，父级没有高度，后面的标准流盒子会上移
    5. 清除浮动带来的影响：
      1. 设置高度
      2. 给父元素设置overflow: hidden，最方便的方式。
      3. clearfix：双伪元素清除法，项目中使用，直接给标签添加类就行。
  8. 定位
    1. 定位的作用：盒子层叠，自由摆放
    2. 定位方式：子绝父相
      1. relative-相对定位：自恋型，相对于自己原有的位置进行移动
        1. 不脱标，占有原来的位置
        2. 元素显示模式不变
      2. absolute-绝对定位：拼爹型，相对于已经定位的父元素（广义父，就近父）进行移动，没有则以浏览器窗口左上角为参照物。子绝父相
        1. 脱标，不占位置
        2. 改变元素的显示模式变为行内块
      3. fixed固定定位：死心眼型定位，相对于浏览器进行定位移动
        1. 脱标，不占位置
        2. 改变元素的显示模式变为行内块
    3. 定位层级关系：默认层级是0，可以通过z-index: 整数;设置定位元素的层级
      1. 布局方式层级关系：标准流<浮动<定位
      2. 定位的层级关系：相对、绝对、固定默认层级相同，但是在HTML中写在下面的元素层级更高，会覆盖上面的元素
